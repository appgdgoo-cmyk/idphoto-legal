본 방침은 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 앱은 사용자의 개인정보를 외부 서버로 전송하지 않으며, 모든 사진 처리는 사용자 기기 내에서만 수행됩니다.

---

## 1. 수집·이용하는 정보

본 앱은 다음 정보를 **사용자의 기기 안에서만 처리**하며, 개발자 또는 제3자에게 전송하지 않습니다.

| 항목 | 처리 방식 | 처리 목적 |
| --- | --- | --- |
| 카메라 영상 (촬영 시점) | 기기 내 처리 | 증명사진 촬영, 얼굴 위치/각도 가이드 |
| 갤러리에서 선택한 사진 | 기기 내 처리 | 사용자가 직접 선택한 사진의 편집 |
| 저장된 사진 파일 | 기기 내 저장 | 사용자가 요청한 출력물 보관 |
| 사진 종류 선호 설정 | 기기 내 저장 (DataStore) | 사용자 편의 — 다음 실행 시 동일 설정 표시 |

본 앱은 **이름·전화번호·이메일·계정·결제정보 등 개인을 식별할 수 있는 정보를 수집하지 않습니다.**

## 2. 권한 사용 안내

| 권한 | 사용 목적 | 거부 시 영향 |
| --- | --- | --- |
| 카메라 (`CAMERA`) | 증명사진 촬영 | 카메라 촬영 기능 사용 불가 (갤러리 가져오기는 가능) |
| 사진/미디어 (`READ_MEDIA_IMAGES`) | 갤러리 사진 가져오기 | 갤러리 가져오기 기능 사용 불가 |
| 외부 저장소 쓰기 (`WRITE_EXTERNAL_STORAGE`, Android 9 이하) | 결과물 저장 | 결과물 저장 위치 제한 |

---

## 3. 사진의 처리·저장·전송

- **얼굴 인식·배경 분리 등 모든 영상 처리는 사용자 기기 내에서 수행됩니다.** Google ML Kit의 온디바이스(on-device) 모델만 사용하며, 영상이나 사진을 외부 서버로 전송하지 않습니다.
- 촬영·편집된 사진은 사용자가 "저장"을 선택할 때에만 기기의 사진 폴더에 저장됩니다.
- 저장된 사진의 사본 또는 사진의 메타데이터는 개발자에게 전송되지 않습니다.
- 사용자가 사진을 다른 앱(예: 메시지, 메일)에 공유하는 경우, 해당 앱의 개인정보 처리방침이 별도로 적용됩니다.

## 4. 제3자 라이브러리 / SDK

본 앱은 다음 온디바이스 라이브러리를 포함합니다. 이들 라이브러리는 본 앱 내에서 **영상이나 개인정보를 외부로 전송하지 않습니다.**

- Google ML Kit Face Detection — 얼굴 위치·각도 인식
- Google ML Kit Subject Segmentation — 인물·배경 분리
- AndroidX CameraX — 카메라 미리보기·촬영
- Jetpack Compose, Coil — 화면 표시·이미지 로딩

광고 SDK, 분석 SDK, 충돌 보고 SDK는 포함하지 않습니다.
